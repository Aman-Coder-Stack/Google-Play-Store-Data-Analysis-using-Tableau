# Google-Play-Store-Data-Analysis-using-Tableau
This project focuses on analyzing Google Play Store applications using Tableau to uncover meaningful insights about app performance, user engagement, and market trends. The dashboard provides an interactive and visual way to explore data related to installs, ratings, reviews, categories, and user sentiment.

🎯 Objectives
Analyze overall app performance across different categories
Compare Free vs Paid applications
Understand user behavior using ratings and reviews
Identify top-performing categories based on installs and app count
Explore sentiment distribution of user reviews
